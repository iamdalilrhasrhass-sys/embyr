# Embir — Shot Lists, B-roll & Trending Sounds

> Shot list détaillé par script. Chaque plan = 1 prise minimum.
> Format vertical 9:16 (1080x1920). 30fps minimum.

## Setup commun (tous scripts)
- **Caméra**: smartphone verticale, trépied ou main stabilisée
- **Lumière**: fenêtre face à toi (jamais dans le dos) OU ring light 18"
- **Fond**: propre, peu chargé. Mur uni ou intérieur soigné.
- **Audio**: micro-cravate OU main très proche du téléphone. Pas d'écho.
- **Écran téléphone**: mode screen recording (iOS: Centre de contrôle > Enregistrement). Désactive notifications (mode Ne pas déranger).

## Trending Sounds — stratégie

> Les sounds trend changent chaque semaine. Vérifier la page "Trends" TikTok Creative Center avant chaque tournage.
> Règle: le VO original performe mieux long terme (pas de droits, pas de disparition de sound).
> Sounds trend = boost court terme uniquement.

| Type de script | Stratégie son |
|----------------|---------------|
| POV / meme (1, 5) | Sound trend si pertinent, sinon VO |
| Éducatif / tip (2, 6, 8) | VO original (clarté prime) |
| Storytelling (3, 9) | VO original obligatoire |
| Inclusif (4, 7) | VO original + lo-fi soft optionnel en fond |
| Transparence (10) | VO original obligatoire |

### Où trouver les trends du moment
1. TikTok Creative Center → Trends → Sounds (filtre: ton pays + 7 derniers jours)
2. Onglet "For You" → note les sounds qui reviennent sur des vidéos dating/tech
3. Repositories: TrendTok app, TokBoard (tokboard.com)

### Sound original Embir (optionnel)
Créer un sound signature court (3-5s, genre lo-fi/indie) à utiliser en fond sur toutes les vidéos VO.
Avantage: reconnaissance de marque + la plupart des vidéos deviennent "original sound" = TikTok pousse.
