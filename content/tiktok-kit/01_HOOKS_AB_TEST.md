# Embir — 30 Hooks A/B Testables (3 par script)

> Le hook = 1.5 premières secondes. C'est 80% de la performance TikTok.
> Tournez TOUJOURS 3 variantes de hook pour le même script. Postez la variante A,
> puis si la vidéo est moyenne, repostez avec B ou C.

## Format optimal du hook
- 1 visuelle frappante (expression, objet, texte géant) dans les 0.3s
- 1 phrase d'accroche courte (5-8 mots) dans les 1.5s
- Jamais "Salut les gars" ou "Aujourd'hui je vais vous parler de" = mort

## Matrice de sélection des hooks

| Objectif | Hook à privilégier |
|----------|-------------------|
| Vues maximales (viralité) | A — direct, intrigue forte |
| Saves / partages | C — éducatif, "tip" |
| Commentaires (engagement) | B — question implicite, controverse douce |
| Conversion (clicks bio) | A ou C — clarté de la proposition |

## A/B test protocol
1. Postez variante A un lundi à 07:30
2. Si < 1000 vues à 6h → variante B le mercredi même créneau
3. Si < 1000 vues → variante C le vendredi
4. La variante gagnante devient le "hook de référence" pour ce script → décliner en 3 sous-variantes
