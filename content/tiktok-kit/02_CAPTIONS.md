# Embir — Captions Chunkées (burned-in ready)

> Captions obligatoires sur CHAQUE vidéo (burned-in, pas auto-caption TikTok).
> Format: chunks de 2-5 mots, une ligne à l'écran, timing approximatif.
> Style recommandé: police bold sans-serif, fond semi-transparent, mot-clé en couleur (#ff5e36).
> Outil gratuit: CapCut (auto-captions + édition manuelle). Ou Captions app.

## Convention de format
```
[0.0s-1.5s] HOOK
[1.5s-3.0s] chunk
...
```
Le mot entre **astérisques** = à mettre en couleur accent (#ff5e36 ember) pour attirer l'œil.

## Notes de style captions
- Police: Montserrat Bold ou Proxima Nova (CapCut > Style > Bold)
- Couleur texte: blanc #FFFFFF
- Couleur accent (mots-clés): #ff5e36 (ember Embir)
- Fond: bande semi-transparent noir 40% derrière le texte
- Position: bas-centre, 15% depuis le bas
- Taille: 40-50pt (lisible sur mobile)
- Animation: pop-in par mot (CapCut > Animation > Spring)
