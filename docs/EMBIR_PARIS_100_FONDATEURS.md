# EMBIR — Plan Paris 100 membres fondateurs

Dernière mise à jour : 2026-06-08

## Objectif

Signer les 100 premiers profils réels sur Paris / Île-de-France avant de relancer de l'automatisation.

Le problème n'est pas le nombre de pages SEO. Le problème est la liquidité : une app de rencontre vide ne retient personne. On concentre donc tout sur une seule poche géographique.

Page à pousser : `https://embir.xyz/paris`

## Règles non négociables

- Pas de faux profils.
- Pas d'automatisation X tant que le compte est rate-limité.
- Pas de spam massif : prospection manuelle, ciblée, polie.
- Pas de promesse “des milliers de membres” tant que ce n'est pas vrai.
- Une seule métrique principale : profils créés avec photo + ville Paris/IDF.

## Offre

**Embir Paris — les 100 membres fondateurs**

- Gratuit au lancement.
- Badge membre fondateur.
- Messages gratuits.
- Influence directe sur les prochaines fonctionnalités.
- Communauté locale réelle, pas une app vide maquillée.

## Routine 7 jours

### Chaque matin

1. Vérifier les inscriptions de la veille.
2. Noter : inscrits, profils complétés, profils avec photo, invitations envoyées.
3. Poster une story ou un post manuel avec `embir.xyz/paris`.
4. Envoyer 20 messages manuels ciblés.

### Chaque soir

1. Répondre à tous les retours.
2. Relancer les inscrits sans photo avec un message humain.
3. Demander à 5 membres d'inviter 2 amis.
4. Noter les objections exactes dans Obsidian.
