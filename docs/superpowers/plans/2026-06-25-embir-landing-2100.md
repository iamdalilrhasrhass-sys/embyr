# Embir Landing 2100 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the current Embir homepage with the approved interactive editorial landing while preserving SSR SEO content, bilingual behavior, accessibility and mobile performance.

**Architecture:** Keep `src/app/[locale]/page.tsx` as a Server Component and compose focused landing chapters. Limit client JavaScript to four isolated instruments: the compatibility compass, reciprocity controls, personal-universe tabs and intentions rail. Use pure TypeScript geometry helpers, SVG/CSS rather than a 3D runtime, and a generated standalone universe image through `next/image`.

**Tech Stack:** Next.js 16 App Router, React 19, TypeScript strict, next-intl routing, Tailwind 4 plus page-scoped CSS, SVG, `next/image`, Node test runner, Puppeteer/Browser QA.
